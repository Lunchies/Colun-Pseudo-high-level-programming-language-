"Colun — Язык, который объясняет себя сам"

Colun — это псевдо-высокоуровневый учебный язык программирования, созданный для того, чтобы новички поняли, 
как работает код, а не просто запомнили команды. Каждая программа выполняется с пошаговыми комментариями, а ошибки превращаются в диалог.
Colun зависит от Python 3.14, и собственно на нем сделан компилятор.


( ❗ Colun не заменяет Python или JavaScript. )

Особенности:

- Объяснение выполнения — интерпретатор рассказывает, что происходит на каждом шаге.
- Диалоговые ошибки — если переменная неизвестна, Colun спросит: «Чему равно X?».
- Простой синтаксис — все команды начинаются с Program, структура — как колонна текста.
- Педагогические метафоры — Forget (забыть), Ask (спросить), Stop (остановить).

Сейчас конкретно Colun находится в стадии продолжения разработки языка, с добавлением новых фич.
Colun - Открытый язык, его компилятор можно переделать, и использовать как вам угодно. (GPL 3.0 лицензия)

Пример программы:

Program Start;
Program call function: Print;
Program call function: Ask;
Program Ask "Как тебя зовут?", name;
Program Print "Привет,", name;
Program if name = "Анна" then Print "Ты — Анна!";
Program End;

Что уже есть в Colun Compiler v1.2 (именно в компиляторе, и какие команды поддерживает, сам Colun release версия 1.1)

Print	- Вывод текста
Ask -	 Ввод данных
Calculate	- Вычисления
Forget -	«Забыть» значение переменной
Sleep -	Пауза (секунды)
Loop -	Повторение блока кода
If/Else	- Условия
And/Or - Логические операторы
Stop -	Досрочное завершение
! - записи в редакторе
